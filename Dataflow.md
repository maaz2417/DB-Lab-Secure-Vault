Basically, it all starts when someone uses the app. When a new user signs up, their account info gets dropped right into the users table. We hash their master password first so it stays safe. While that's happening, the database automatically spins up a row in the user_preferences table with some default settings. On a day-to-day basis, when you type in a new password or bank account to save, our Flask backend grabs it, scrambles it up with AES-256 encryption, and shoves it into either the vault or bank_credentials table depending on what kind of account it is.

How Data Moves Around (Processing)
Once the data is in, moving it around is all about keeping you logged in and checking security rules. When you log in, the backend double-checks your password against the users table. If you're good, it creates a temporary token and saves it in the sessions table so the app remembers you.

Now, every time you actually do something in the app—like clicking to view or edit a password—the backend quickly peeks at the credential_types table. It does this to know the rules, like if it needs to clear your clipboard extra fast because you're looking at a bank password. The best part of the flow is that every single time you do something, the system quietly writes a note in the audit_log table. It ties your user ID to the vault ID so there's a permanent receipt of what went down. Oh, and if you ever update a password, we don't just delete the old one; we slide the old encrypted string over to the password_history table for archiving.

What Comes Out (Output)
The main thing the app spits out is your decrypted passwords so you can actually read them. When you click on a saved login, the app hits the vault table, pulls out that scrambled AES-256 string, unlocks it on the backend since you have an active session, and then shows the plain text password on your screen. Besides that, the other outputs are the dashboard itself—which just pulls a list of all the website names tied to your user ID—and the history page, which pulls from the audit log so you can see exactly what you've been doing with your account recently.
